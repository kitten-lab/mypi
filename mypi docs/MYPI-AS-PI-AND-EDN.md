**Status:** **LOCKED** 2026-07-23  
**Pace note:** Rome wasn’t built in a day or 14 months. EXTRAORDINARY kisses are not a sprint milestone.

---

## The reframe (locked)

**MYPI** = **My P.I.** — *private investigator*  
Terminal stations, backrooms, startup functions (mailroom, early tooling), glass/corpus, messy OK.  
Port later into EDN **with classy names** when a surface graduates.  
Not a failed public SaaS. Not the world city.

**EDN** = **go.edn** — ROM games, imported cities, sky language, public destinations.  
Slow. Intentional. Link to MYPI PI surfaces; don’t clone the office into every door.

**Link, don’t clone:** EDN shells point at MYPI investigator surfaces until a real port is earned.

## Split of labor

| | **MYPI (P.I.)** | **EDN (public / world)** |
|--|-----------------|---------------------------|
| **Job** | Investigate, hold glass, run terminals, backrooms, private myth-in-code salvage | Destinations, ROMs, cities, sky language, public play |
| **Audience** | You (and chosen witnesses) | Players / visitors / “the network” |
| **Alphabet** | May still carry old scars; migrate place language as you touch files | **New alphabet only** (no sys/dom/mod story in new names) |
| **Pace** | Can stay messy if *honestly private* | Slow, intentional, bit-by-bit |
| **Storage** | Source of truth for investigator data, glass, terminals | Links + thin public skins; port heavy data when ready |
| **What “done” means** | Not “perfect platform” — **working backrooms** | Not “port all MYPI” — **one honest door at a time** |

---

## How a thing gets made (default pipeline)

```
1. Need / myth / game idea
2. Build or keep investigator surface in MYPI
   (terminal, backroom tool, glass shelf, ledger)
3. When public-facing: EDN destination / ROM / city block
   → shell + sky language + **link** to MYPI surface
4. Only port storage/code when the public thing
   actually needs its own copy (and names are right)
```

**Excitement-to-online** stays on the **EDN** side and is throttled.  
**Excitement-to-know** stays on the **PI** side (logs, shape, terminals).
